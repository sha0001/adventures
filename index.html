Definitions

This section of the page will be for defining the 'grammatical' terms for general structural elements in HTML and CSS.
This is distinguished from lists of specific tags or attributes in that it is designed to give a higher level view of the structure of the language

HTML

  • Tags - Are placed at the start and end of an element. Opening tags are surrounded by < >, while closing tags are displayed as </>.
    Some tags are open and do not require a closing tag. List of Tags by function
  • Attributes - Attributes allow you to customise a tag, and are defined within the opening tag List of Attributes
  • Elements - An element is everything contained within two tags

CSS

  • Selector - points to the html element you want to style. The tag heading without < >)
  • Property - The property of the element that you intend to change eg. Color List of Properties
  • Value - The value you want the property to become. Can be a numerical or descriptive value eg. "Blue"
  • Declaration - A property and a Value, separated by :, finished with ;
  • Declaration Block - A series of declarations contained within curly brackets. eg. {color:blue;font-size:12px;}
  • Functions - A function completes a task based on predetermined inputs. Arguments are placed within brackets after calling the function. To be used as values for properties.
    eg. linear-gradient(red, yellow, green) List of functions

Required HTML

<!DOCTYPE html> Defines which version of HTML is to be used.

<html> Marks that the following is in HTML

<head> The 'head' tag is a container for metadata. It is not displayed.

<title></title> The Title of the page - displayed in Tabs and Search results

<link rel="stylesheet" type="text/css" href="mystyle.css"> Invokes CSS style sheet

<meta name="viewport" content="width=device-width, initial-scale=1.0"> Scaling for various devices


</head>

<body> Marks the start of visible content

</body>

</html>

In addition to the above tags, the html file needs to be saved as the .html file type.

Invoking CSS Style Sheet

CSS can be invoked in three different ways:

  • In the HTML document header,placed between the <style> tags:
    h1 {color: maroon; margin-left: 40px;}
  • Inline in the HTML document, within the "style" attribute of the relevant element
    <h1 style="color:blue;text-align:center;">This is a heading</h1>
  • In a separate style sheet, placing the following element between the <head> tags
    <link rel="stylesheet" type="text/css" href="mystyle.css"> link

External style sheets are to be saved as the .css file type.

Multiple CSS sheets can be used simultaneously, but the most recently read will be used.
